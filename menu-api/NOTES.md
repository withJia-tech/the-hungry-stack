# menu-api — Spring Boot

## The exercise
Build a `/menu` endpoint. Open it in a browser. Get back the 5 dishes from `../data/dishes.json` as JSON.

## What you'll have to figure out
- How does Spring Boot start a web server?
- Where do you put the route?
- How do you read a JSON file from disk?
- How do you turn a Java list into a JSON response?

## You'll know it works when
You type `http://localhost:8080/menu` in your browser and see all 5 dishes.
